# Blinkit Grocery Sales Dashboard 📊

## 🎯 Objective

To analyze Blinkit's grocery sales data and build an interactive dashboard that provides meaningful business insights and helps track overall sales performance.

## 📂 Raw Data

The project contains raw grocery sales data, including:

- Item categories
- Outlet details
- Sales figures
- Ratings
- Establishment information

## 📌 KPIs

- 💰 Total Sales: $1.20M
- 💵 Average Sales: $141
- 📦 Number of Items: 8523
- ⭐ Average Rating: 3.9

## 📈 Visualizations

- Outlet Establishment Trend Analysis
- Fat Content Distribution
- Item Type Analysis
- Fat by Outlet Analysis
- Outlet Size Distribution
- Outlet Location Analysis
- Interactive Filters (Outlet Size, Outlet Location Type, Item Type)

## 💡 Business Insights

- Identified top-performing outlet locations.
- Analyzed sales distribution across outlet sizes.
- Compared sales performance across different item categories.
- Evaluated customer ratings and outlet performance trends.

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Analysis
- Data Visualization
